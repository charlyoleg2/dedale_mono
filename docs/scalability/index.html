SvelteKit / Hono integration

Nodejs enables running the same javascript code on the server and in the browser. This enables frontend frameworks like svelteKit where SSR and CSR can be easily exchanged. If the backend framework (e.g. hono) is also in javascript, it can be integrated/imported into the frontend framework.
