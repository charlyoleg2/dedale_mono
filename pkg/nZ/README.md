backend node nZ
===============

Presentation
------------

This package contains the backend node *nZ*.


Dev
---

```bash
git clone https://github.com/charlyoleg2/dedale_mono
cd dedale_mono
npm i
npm -w nZ run clean
npm -w nZ run ci
npm -w nZ run run_n
open http://localhost:3010/doc
open http://localhost:3010/swagger
npm run test_nZ
```
